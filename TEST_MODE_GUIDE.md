# 测试模式使用指南

## 概述

测试模式允许你单独控制每个电机，验证角度定义和运动范围。

## 切换到测试模式

### 方法1：文本命令（推荐）
```
mode test
```

### 方法2：VOFA二进制命令
发送HEX命令：`FF 31 00 00 00 00 00`

## 控制单个电机

### 方法1：文本命令（推荐，简单）

**格式**：`set <motor_id> <angle>`

**示例**：
```
set 1 45      # 控制底座转到45度
set 2 30      # 控制大臂转到30度
set 3 -20     # 控制小臂转到-20度
```

**电机ID说明**：
- `1` - 底座旋转电机（theta0）
- `2` - 大臂电机（theta1）
- `3` - 小臂电机（theta2）

### 方法2：VOFA二进制命令

**格式**：`FF 30 <motor_id> <angle_float_4bytes>`

**示例**（控制电机1转到30度）：
```
FF 30 01 00 00 F0 41
```
其中 `00 00 F0 41` 是30.0的IEEE 754小端序表示。

## 其他文本命令

### 切换工作模式
```
mode auto     # 切换到自动模式
mode test     # 切换到测试模式
mode follow   # 切换到跟随模式
```

### 控制气泵
```
pump on       # 打开气泵
pump off      # 关闭气泵
```

### 软件复位
```
reset         # 触发软件复位
```

### 帮助信息
```
help          # 显示所有可用命令
```

## 使用步骤

### 1. 连接串口
- 打开串口助手（VOFA、PuTTY、Tera Term等）
- 波特率：115200
- 数据位：8
- 停止位：1
- 校验：无

### 2. 切换到测试模式
```
mode test
```

### 3. 测试各个电机
```
set 1 0       # 底座回到0度
set 2 90      # 大臂垂直向上
set 3 0       # 小臂水平
```

### 4. 验证角度定义

**测试theta1（大臂）**：
```
set 2 0       # 水平向前
set 2 45      # 向上45度
set 2 90      # 垂直向上
set 2 -30     # 向下30度（如果允许）
```

**测试theta2（小臂）**：
```
set 3 0       # 水平
set 3 30      # 向上30度
set 3 -30     # 向下30度
```

**测试theta0（底座）**：
```
set 1 0       # 正前方
set 1 90      # 向左90度
set 1 -90     # 向右90度
```

## 角度定义说明

### theta0（底座旋转）
- 0° = 正前方
- 正值 = 逆时针旋转（从上往下看）
- 负值 = 顺时针旋转

### theta1（大臂）
- 0° = 水平向前
- 正值 = 向上抬起
- 90° = 垂直向上

### theta2（小臂）
- **绝对角度**（相对于水平面）
- 0° = 水平
- 正值 = 向上
- 负值 = 向下

**注意**：你观察到的现象是正确的！
- 给电机3（小臂）**正角度** → 小臂**下压**
- 给电机3（小臂）**负角度** → 小臂**上抬**

这说明你的电机安装方向与代码定义相反。但这**不影响计算**，因为：
1. FK正解会根据你给的角度计算末端位置
2. IK逆解会根据目标位置计算需要的角度
3. 只要保持一致，系统就能正常工作

如果你想让正角度=上抬，可以在发送给电机时取反，但建议保持现状，避免混乱。

## 安全注意事项

1. **测试前确认**：
   - 机械臂周围无障碍物
   - 电机电源已连接
   - 机械臂处于安全位置

2. **角度限制**：
   - 不要超出机械臂的物理限制
   - 注意大臂和小臂的夹角限制（30°-150°）

3. **紧急停止**：
   - 如果运动异常，立即断电
   - 或发送 `reset` 命令复位系统

4. **测试顺序**：
   - 先测试小角度变化
   - 确认运动方向正确后再测试大角度

## 故障排查

### 命令无响应
- 检查串口连接
- 确认波特率为115200
- 检查是否在测试模式（`mode test`）

### 电机不动
- 检查电机电源
- 检查电机ID是否正确（1=底座, 2=大臂, 3=小臂）
- 查看串口是否有错误信息

### 角度方向相反
- 这是正常的，取决于电机安装方向
- 不影响系统功能，FK和IK会自动处理
- 建议保持现状，不要修改

## 示例测试流程

```
# 1. 切换到测试模式
mode test

# 2. 回到初始位置
set 1 0       # 底座
set 2 90      # 大臂垂直
set 3 0       # 小臂水平

# 3. 测试大臂运动
set 2 60
set 2 45
set 2 30

# 4. 测试小臂运动
set 3 30
set 3 0
set 3 -30

# 5. 测试底座旋转
set 1 45
set 1 -45
set 1 0

# 6. 回到自动模式
mode auto
```

## 命令对比

| 功能 | 文本命令 | 二进制命令 |
|------|---------|-----------|
| 控制底座 | `set 1 30` | `FF 30 01 00 00 F0 41` |
| 控制大臂 | `set 2 90` | `FF 30 02 00 00 B4 42` |
| 控制小臂 | `set 3 -20` | `FF 30 03 00 00 A0 C1` |
| 测试模式 | `mode test` | `FF 31 00 00 00 00 00` |
| 自动模式 | `mode auto` | `FF 32 00 00 00 00 00` |
| 跟随模式 | `mode follow` | `FF 33 00 00 00 00 00` |
| 软件复位 | `reset` | `FF 3F 00 00 00 00 00` |

**推荐使用文本命令**：更直观、更容易记忆、不需要计算float编码。
