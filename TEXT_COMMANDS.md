# 文本命令快速参考

## 快速开始

直接在串口助手中输入文本命令，按回车发送。

## 所有命令

### 电机控制
```bash
set <motor_id> <angle>    # 控制单个电机
```
示例：
- `set 1 45` - 底座转到45度
- `set 2 30` - 大臂转到30度
- `set 3 -20` - 小臂转到-20度

### 模式切换
```bash
mode <auto|test|follow>   # 切换工作模式
```
示例：
- `mode test` - 测试模式（单独控制电机）
- `mode auto` - 自动模式（视觉抓取）
- `mode follow` - 跟随模式（实时跟随）

### 气泵控制
```bash
pump <on|off>             # 控制气泵
```
示例：
- `pump on` - 打开气泵
- `pump off` - 关闭气泵

### 系统命令
```bash
reset                     # 软件复位
help                      # 显示帮助信息
```

## 常用操作

### 测试单个电机
```bash
mode test      # 切换到测试模式
set 2 30       # 控制大臂到30度
set 3 -20      # 控制小臂到-20度
mode auto      # 切回自动模式
```

### 手动抓取测试
```bash
mode test      # 测试模式
pump on        # 打开气泵
pump off       # 关闭气泵
```

### 快速复位
```bash
reset          # 软件复位系统
```

## 电机ID对照

- `1` = 底座旋转（theta0）
- `2` = 大臂（theta1）
- `3` = 小臂（theta2）

## 注意事项

1. 命令不区分大小写
2. 参数之间用空格分隔
3. 按回车键发送命令
4. 支持负数角度（如 `-20`）
5. 支持小数角度（如 `30.5`）

## 兼容性

文本命令和VOFA二进制命令可以混用，系统会自动识别。
